# Frontend — CampusConnect 360

SPA en **React**. Contiene los 3 portales operables y el dashboard directivo, todos
consumiendo las APIs a través del **API Gateway** (`http://localhost:8080`).

- **Portal Académico** (rol SECRETARIA)
- **Portal Financiero** (rol FINANZAS)
- **Portal Docente / Bienestar** (rol DOCENTE)
- **Dashboard Directivo** (rol DIRECCION) — alimentado por `/analytics/*`
- **Puerto:** 3000

> Se implementa en el **Paso 7** del plan. Aquí irá el proyecto React (`package.json`, `src/`, `Dockerfile`).
